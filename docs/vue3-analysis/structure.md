# 源码目录

```bash
packages
├── compiler-core // 核心编译器（与平台无关）
├── compiler-dom // dom编译器
├── compiler-sfc // vue单文件编译器
├── compiler-ssr // 服务端渲染编译
├── global.d.ts // ts声明文件
├── reactivity // 响应式模块
├── reactivity-transform
├── runtime-core // 运行时核心代码（平台无关）
├── runtime-dom // 运行时dom相关api
├── runtime-test // 运行时测试
├── server-renderer // 服务端渲染
├── sfc-playground // 单文件在线调试器
├── shared // 内部共享工具库
├── size-check // 简单应用，用来测试代码体积
├── template-explorer
├── vue // vue入口文件
└── vue-compat
```
