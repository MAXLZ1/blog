TIP

back，回退一个历史记录，相当于go(-1)。

forward，前进一个历史记录，相当于go(1)。

back、forward都是通过go实现。关于go的实现可参考：go

const router = {
  // ...
  back: () => go(-1),
  forward: () => go(1),
  // ...
}
