replace #

TIP

通过替换历史记录，导航至一个新的URL。

replace与push作用几乎相同，如果push时指定replace: true，那么和直接使用replace一致。

function replace(to: RouteLocationRaw | RouteLocationNormalized) {
  return push(assign(locationAsObject(to), { replace: true }))
}
